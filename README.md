This is the list of all my coding projects and coding challenges (Well almost all)

## Description

This repository contains the HTML and assets for the FutureTech website, a fictional technology company's online presence. The website showcases the latest developments and products of FutureTech, featuring responsive styling using Bootstrap and modern animations using AOS. Some icons from Font Awesome were used.

## Main Page

The Main Page aims to introduce FutureTech, its products, and its core values.

## About Page

The About Page provides an overview of FutureTech, emphasizing its mission, vision, values, and contributions.

## Features Page

The Features Page describes the innovative technologies offered by FutureTech.

## Contact Page

The Contact Page provides users with information on how to contact FutureTech and a form to send messages, questions, or complaints.

## User Page

The User Page allows users to manage their profile, appearance, privacy, and security settings. Users can update their profile information, select avatars, themes, and fonts, and manage privacy and security settings.

## Layout.html

This code provides the basic structure and layout for the web page and uses the Bootstrap framework and Font Awesome for styling and icons.

## Footer

The footer contains essential company information, links to various pages, and social media icons. It also credits Nico Viljoen (the creator).

## Images

This file contains all of the images used in the project. Most of the images were retrieved from Google Images.

## CSS Themes

These stylesheets are well-organized and cover various aspects of a website's design, including responsiveness for smaller screens. There are also two different CSS stylesheets, one for the Default theme and the other for the Dark theme.

## JavaScript

The provided JavaScript code is a client-side script for the web application. It performs several functions related to updating the app's theme, font, and user profile, and it also includes functions for making AJAX requests to the server.

## Flask Implementation

The website has been integrated with Flask, a web framework, to enable dynamic content and interactivity. The Flask implementation enhances the user experience and simplifies backend management.

## FutureTech Database

This project uses a relational database named "futuretech.db" to store and manage various types of data related to the website's style and some personal information. The database is built using SQLite and consists of two tables, including:

- `users`: Stores user information.
- `appearance`: Stores the user's web style.

### Technologies Used

- **Database Management System**: SQLite
